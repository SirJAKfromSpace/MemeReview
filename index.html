Data Visualization and Clustering of Similar Memes

This page visualizes a large image collection within an interactive WebGL scene. Each image was processed with an Inception Convolutional Neural Network, trained on a Kaggle Dump of meme images, similar images were clustered using TSNE, and projected onto a two-dimensional manifold with the D3.js library such that similar images appear nearby one another.
